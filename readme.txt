
/****Clone Github repository and Create a remote repository****/ 

In the command-line shell:

<code>
	
	git clone git@github.com:colinjlacy/idea-feast.git
	cd idea-feast
	git remote rm origin

	bundle install

	rails new [application name]
</code>

Add the model, view, and controller

<code>
	rails g scaffold [route name???]
	
</code>

Create the database
<code>

	rake db:migrate
</code>


<code>
	rake db:setup
	
	
	
<code>
